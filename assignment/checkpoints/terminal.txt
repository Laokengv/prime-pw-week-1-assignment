# 1 - What is the command for listing the files in the current folder (directory)?
Use command 'ls'

# 2 - What is the command for changing directories?
Use command 'cd'

# 3 - What is the command for going "up" a folder (to a parent directory)?
 Use commands 'cd .. or cd ~'

# 4 - What is a quick way to create a new folder?
Use command 'mkdir' to create a new folder

# 5 - What output would you expect when you run the command 'ls' in an empty folder? What about a folder containing files?
When you run the command 'ls' in an empty folder nothing will happen. When you use 'ls' in a folder that contains files it will list all the files within that folder

# 6 - What is a quick way to create a new file?
Use command 'touch' to create a new file

Your terminal will tell you what the latest version your git is running

# 8 - Why would you run the command 'git status'?
The 'git status' command displays any changes you've made on git that have not been committed yet

# 9 - What does the command 'git add .' do?
The 'git add' command adds new or changed files to git 

# 10 - What is the command to push your committed changes to your repo on GitHub?
Use command 'git push' to push your committed changes
